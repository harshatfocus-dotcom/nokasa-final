# NoKasa - Responsible Clothing Recycling

NoKasa connects households with responsible recycling and resale networks with a convenient doorstep pickup. This project is built with Next.js, Framer Motion, and Tailwind CSS, following premium minimalist design principles inspired by Apple.

## 🚀 Live Demo

Check out the live deployment on Vercel:
**[https://nokasav2.vercel.app](https://nokasav2.vercel.app)**

## ✨ Key Features

- **Apple-Style Hero Section**: Immersive animations with magnetic interactions and multi-layered layout.
- **Service Checker**: Redesigned, minimal area-readiness section with a procedural 3D-transforming Bangalore network map.
- **Dynamic Leaf Background**: Ambient particles and organic movement to reflect ecological sustainability.
- **Deep Clean Circular Economy**: A full suite of sections documenting 'The Crisis', 'The Solution', and 'Project Impact'.
- **Mobile Optimized**: Fully responsive across all devices with official App Store and Google Play connectivity.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev)
- **Map Visuals**: [Dotted Map](https://github.com/mavo/dotted-map)
- **Deployment**: [Vercel](https://vercel.com)

## 📦 Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
